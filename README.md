# Projet Fin Année

## Structure du projet

- `/*.c` : Code source principal en C
- `export/` : Fichiers CSS, JS et HTML
- `Makefile` : Automatisation de la compilation
- `ressources/` : Fichier Data
- `/README.md` : Ce fichier

## Utilisation

1. **Compiler le projet**
    - `make` : compile le projet 
    - `make web` : supprime tous les fichiers HTML (s'ils existent) et exécute l'algo
    - `make hsup` : supprime tous les fichiers HTML (s'ils existent)

2. **Lancer le programme**
    - `./projet.exe --file <nom fichier>` : génère les pages HTML
    - Ouvrir `home.html` dans un navigateur pour commencer l'aventure
